# Youtube Consumer

Youtube Consumer lets you download videos/or just audio tracks from youtube. This is a remake of my old project.

## Installation

```bash
git clone
cd youtube-consumer
npm install
```

## Usage

```bash
npm start
```

## Template

This project uses [this](https://github.com/paolodelfino/Node.JS-Template) as a template.
